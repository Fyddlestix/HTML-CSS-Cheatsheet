HTML Cheatsheet

Element Description

Important elements

<!DOCTYPE html>

The first element in an html document. It informs the browser it is parsing an HTML5 document and prevents errors or quirks when rendering the page.

<html>

The root element. All other elements must be descendants of the <html> element.

<head>

Contains metadata relating to the page.

<link>

used to link external resources to the webpage such as favicons, stylesheets and fonts.

<body>

All content rendered by the browser is contained within this element. There can only be one <body> element in an HTML document

Basic Elements

<h1> - <h6>

Headings. In order of importance, with <h1> being the most important and <h6> being the least.

<p>

The paragraph element represents block-level grouping of related content.

Non-Semantic Elements

<div>

A generic container for flow conent. Mainly used for styling with CSS.

Semantic Elements

<header>

Represents introductory conduct such as navagation bars, logos and other elements.

<nav>

Represents a section of a page used to provide navigational links.

<main>

Represents the main content of a webpage contained within the <body> element.

<footer>

Represents a content that forms the footer of a webpage.

<section>

Represents a standalone thematic section of content on a web page.

<article>

Represents a self contained composition in a document, page, application or site.

<aside>

Contains content that represents a portion of content indirectly related to the main content.

<figure>

Encapsulated all types of media.

<figcaption>

Used to describe the media contained in the <figure> element.

<video>

Embeds a media player which supports video playback.

<audio>

Embeds sound content in the document.

<img>

Embeds an image into the document.

Tables

<table>

Used to represent tabular data with a two dimensional table comprised of rows and columns which contain data.

<thead>

Defines a set of rows containing the head of the table.

<th>

The table header element defines a cell as a header.

<tbody>

Contains all elements that make up the main body of the table.

<tr>

Defines a row of cells in a table.

<td>

Defines a cell of table data.

<tfoot>

Defines tables rows that make up the footer of the table.

Forms

<form>

Represents a document section containing interactive controls for submitting information

<input>

Used to create interactive controls for web-based forms. The <input> element is one of the most powerful and complex in all of HTML due to the sheer number of combinations of input types and attributes.

<select>

Represents a control that provides a menu of options using the <option> element.

<datalist>

Used to contain a set of <option> elements that represent the permissible or recommended options available.

<textarea>

Represents a multi-line plain-text editing control.
