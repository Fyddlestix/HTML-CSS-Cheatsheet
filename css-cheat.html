CSS Cheatsheet

Selector/Rule Syntax Description
Selectors
Universal Selector * {

} 		Selects all elements.
Type Selector span {

} 		Selects all span elements.
Class Selector .introduction {

} 		Selects all elements that have the "introduction" class.
ID Selector #menu-button {

} 		Selects the element that has the "menu-button" id.
Attribute Selector [controls] {

} 		Selects all elements that have the controls attribute.
Grouping Selectors
List Selector span, div {

} 		Selects all <span> and <div> elements.
Combinators
Descendant Combinator div p {

} 		Selects all <p> elements that are nested inside a <div> element.
Child Combinator ul > li {

} 		Selects all <li> that are nested directly inside a <ul> element.
General Sibling Combinator p ~ span {

} 		Selects all <span> elements that follow a <p> element.
Adjacent Sibling Combinator h2 + p {

} 		Selects the first <p> element that immediately follows an <h2> element.
Column Combinator col || td {

} 		Selects all <td> elements that belong to the scope of the <col>
Pseudo-Classes and Pseudo-Elements
Pseudo-Classes a:hover {

} 		The : pseudo allow the selection of elements based on state information. a:hover selects all <a> elements when the user mouses over them.
Pseudo-Elements p::first-line {

} 		Represents entities that are no included in HTML. p::first-line will select the first line of all <p> elements.
Visual Rules
Color color: lightblue; Sets the foreground color of an element's text.
Background Color background-color: red; Sets the background color of an element.
Opacity opacity: 0.5; Sets the opacity of an element with 0 being transparent and 1 being opaque.
Background Image background-image: url(./imagelocation); Sets the background image on an element.
!important color: red !important; Marks the declaration as important. This will override all other rules regardless of specificity.
The Box Model
Parts Description Rules
Content Box The area where your content is displayed sized using: inline-size and block-size and width or height
Padding Box The padding sits around the content as white space. sized using padding
Border Box The border box wraps the content and any padding Sized using border
Margin Box The margin is the outermost layer, wrapping the content, padding and border as whitespace between this box and other elements. Sized using margin
Display and Positioning
Selector/Value Syntax Description
Position position: value; Sets how an element is positioned in a document.
Relative position: relative; The element is positioned according to the normal flow of the document then offset raltive to its default position.
Absolute position: absolute; The element is removed from the normal flow and no space is created for the element. It is positioned relatice to its closest positioned ancestor.
Fixed position: fixed; The element is removed from the normal flow and no space is created. The element is fixed in position relative to its initial containing block.
Sticky position: sticky; The element is positioned according to the normal flow of the document and then offset relative to its nearest scrolling ancestor and containing block.
Color
Model Syntax Description
Hex Color #FFFFFF Hexidecimal is a number system. Ranges from 0 - 9 and A - F starts with the # symbol.
rgba rgba(255, 60, 40, 0.5); Within the parentheses is one value for red, one for green, one for blue, and one for alpha values for color range from 0 - 255, and alpha ranges from 0 - 1.
hsla hsla(180, 50%, 50%, 0.5); The values in the parentheses are Hue, Saturation, lightness and alpha. Hue is the angle on the color wheel.
Named Color color: orange; You can select from CSS' limited named colors.
Typeography
Rule Syntax Description
Font Weight font-weight: bold; Sets the weight (thinckness) of the font.
Font Style font-style: italic; Sets the style of the font with either normal, italic, or oblique
Line Height line-height: 1.5; Sets the height of a line box.
Word Spacing word-spacing: 2px; Sets the length of space between words and tags.
Letter Spacing letter-spacing: 2px; Sets the length of space between individual characters.
Text Align text-align: center; sets the horizontal alignment of in-line level elements.
@Font Face @font face {

font-family: sans;

src:url("./fonts/font.woff") format("woff");

} 		Specifies a custom font with which to display text.
